# SpawnDev.Identification
[![NuGet](https://img.shields.io/nuget/dt/SpawnDev.Identification.svg?label=SpawnDev.Identification)](https://www.nuget.org/packages/SpawnDev.Identification) 

Identity verification using ECDSA signatures for browser and non-browser platforms.

## Installation

```bash
dotnet add package SpawnDev.Identification
```

## Overview

SpawnDev.Identification provides ECDSA-based identity verification that works in both Blazor WebAssembly (browser) and standard .NET (desktop/server). Sign and verify data to prove identity across platforms.

## Targets

.NET 8.0, 9.0, 10.0
